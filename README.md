# HeroUI Portfolio Site

A modern, fast web application built with **Vite**, **HeroUI**, and **TypeScript**.

## Stack

- **Vite** - Next generation frontend tooling
- **TypeScript** - Type-safe JavaScript
- **HeroUI v2** - Beautiful React components
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **Error Tracking** - GlitchTip integration

## Quick Start

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Environment Setup

Create a `.env` file in the root directory:

```env
VITE_GLITCHTIP_DSN=your-glitchtip-dsn-here
```
